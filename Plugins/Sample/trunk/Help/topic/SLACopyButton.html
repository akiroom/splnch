SLACopyButton

ボタンをクリップボードにコピーします。

構文

function SLACopyButton(ID, GroupIndex, ButtonIndex: Integer): BOOL; stdcall;

引数

ID ボタンをコピーするパッドの ID
GroupIndex コピーするボタンがあるボタングループのインデックス
ButtonIndex コピーするボタンの位置を示すインデックス

戻り値

関数が成功したら TRUE 、失敗したら FALSE を返します。

解説

既存のボタンをクリップボードにコピーします。
