SLACopyGroup

ボタングループを複製を作成します。

構文

function SLACopyGroup(ID, GroupIndex, NewIndex: Integer): BOOL; stdcall;

引数

ID 複製するボタングループがあるパッドの ID
GroupIndex 複製元のボタングループのインデックス
NewIndex 複製された新しいボタングループを挿入する位置のインデックス

戻り値

関数が成功したら TRUE 、失敗したら FALSE を返します。

解説

既存のボタングループの複製を作成し、引数 NewIndex の示す位置へ挿入します。
