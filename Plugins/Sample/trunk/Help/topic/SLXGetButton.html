SLXGetButton

プラグインに含まれるボタンの種類を返します。

構文

function SLXGetButton(No: Integer; ButtonInfo: PSLXButtonInfo): BOOL; stdcall;

引数

No ボタン番号
ButtonInfo ボタン情報を入れる構造体 TSLXButtonInfo へのポインタ

戻り値

引数 No の番号のボタンがある場合には TURE 、ない場合には FALSE を返します。

解説

プラグインの起動時に呼ばれます。

SL4 は引数 No を 0 から順に、この関数が FALSE を返すまで繰り返し実行します。

引数 ButtonInfo が指す構造体 TSLXButtonInfo にボタンの情報を返してください。

SL4 の対応

4.0.0 Build 191 2000/2/2 以降
