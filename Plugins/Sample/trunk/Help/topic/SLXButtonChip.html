SLXButtonChip

ボタンのチップに表示する文字列を返します。

構文

function SLXButtonChip(No: Integer; Value: PChar; Size: Word): BOOL; stdcall;

引数

No 更新するボタンの番号
Value チップに表示する文字列を返すバッファを指すポインタ
Size Value の指すバッファのサイズ

戻り値

関数が成功したら TRUE 、失敗したら FALSE を返します。

解説

関数 SLXGetButton で独自チップを持つボタンとした場合に、ボタンのチップが表示される際に呼ばれます。

引数 Value が指すバッファにプラグインの名前を返してください。バッファには、引数 Size が示すサイズ以上は入れないようにしてください。バッファーオーバーフローを起こします。
