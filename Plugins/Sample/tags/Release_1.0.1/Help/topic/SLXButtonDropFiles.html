SLXButtonDropFiles

ボタンにファイルをドラッグ＆ドロップされると呼ばれます。

構文

function SLXButtonDropFiles(No: Integer; hWnd: HWND; Files: PChar): BOOL; stdcall;

引数

No ファイルをドラッグ＆ドロップされたボタンの番号
hWnd ファイルをドラッグ＆ドロップされたボタンがあるパッドのウィンドウハンドル
Files ファイル名へのポインタ

戻り値

関数が成功したら TRUE 、失敗したら FALSE を返します。

解説

ボタンにファイルをドラッグ＆ドロップされると呼ばれます。

複数のファイルの場合、引数 Files の指すファイル名は、改行（CR+LF）で区切られて渡されます。

TRUE を返すと、ボタンが再描画されます。
