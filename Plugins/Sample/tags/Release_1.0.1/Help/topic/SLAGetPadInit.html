SLAGetPadInit

パッドのプロパティを 1 つ取得します。

構文

function SLAGetPadInit(ID: Integer; Key: PChar; Buf: PChar; BufSize: Integer): BOOL; stdcall;

引数

ID プロパティ取得するパッドの ID
Key パッドのプロパティの名前
Buf プロパティの内容を入れるバッファ
BufSize バッファのサイズ

戻り値

関数が成功したら TRUE 、失敗したら FALSE を返します。

解説

パッドのプロパティを 1 つ取得します。

取得するプロパティは、引数 Key にパッドの設定ファイル PadX.ini と同じ名前を指定します。

取得したプロパティは引数 Buf に文字列として格納されます。
