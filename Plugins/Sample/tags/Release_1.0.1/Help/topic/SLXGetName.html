SLXGetName

プラグインの名前を返します。

構文

function SLXGetName(Name: PChar; Size: Word): BOOL; stdcall;

引数

Name 名前を返すバッファを指すポインタ
Size Name が指すバッファのサイズ

戻り値

関数が成功したら TRUE 、失敗したら FALSE を返します。

解説

SL4 起動時に 1 回だけ呼ばれます。

引数 Name が指すバッファにプラグインの名前を返してください。バッファには、引数 Size が示すサイズ以上は入れないようにしてください。バッファーオーバーフローを起こします。

プラグインの名前は他のプラグインと重複することはできません。ユニークな名前を返すようにしてください。日本語の名前（Shift-JIS）も使えます。

プラグインは必ずこの関数を実装してください。

SL4 の対応

4.0.0 Build 191 2000/2/2 以降
