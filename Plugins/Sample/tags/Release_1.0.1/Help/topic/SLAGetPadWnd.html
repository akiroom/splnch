SLAGetPadWnd

パッドの ID からパッドのウィンドウハンドルを取得します。

構文

function SLAGetPadWnd(ID: Integer): HWND; stdcall;

引数

ID ウィンドウハンドルを取得するパッドの ID

戻り値

パッドのウィンドウハンドルを返します。

解説

パッドの ID からパッドのウィンドウハンドルを取得します。

隠れている状態のパッドは別のウィンドウになります。隠れているパッドのウィンドウハンドルを取得するには SLAGetPadTabWnd を使う必要があります。
