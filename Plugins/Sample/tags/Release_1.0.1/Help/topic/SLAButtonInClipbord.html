SLAButtonInClipbord

クリップボードにボタンで貼り付けられるデータがあるかを返します。

構文

function SLAButtonInClipbord: BOOL; stdcall;

引数

なし

戻り値

クリップボードにボタンで貼り付けられるデータがあれば TRUE 、なければ FALSE を返します。

解説

クリップボードを調べ、ボタンとして貼り付けられるかを確認します。

SL4 からコピーしたボタンのほかに、ファイルやインターネットのショートカットも貼り付けられます。

SL4 の対応

4.0.0 Build 225 2000/12/22 以降
