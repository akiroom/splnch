SLXButtonOptions

プラグインボタンの設定ダイアログボックスを表示します。

構文

function SLXButtonOptions(No: Integer; hWnd: HWND): BOOL; stdcall;

引数

No 設定するボタンの番号
hWnd 設定ダイアログボックスの親ウィンドウになるウィンドウのハンドル

戻り値

設定を変更したら TRUE 、設定を変更しなければ FALSE を返します。

TRUE を返した場合、SL4 は関数 SLXButtonUpdate を実行します。

解説

SL4 の［ボタンの追加／変更］ダイアログボックスで、プラグインの設定を行う際に呼ばれます。

プラグインボタンの設定ダイアログボックスを表示して、そのボタンに関する各種設定を行ってください。

TRUE を返すとボタンの再描画のために関数 SLXButtonUpdate が呼ばれます。
