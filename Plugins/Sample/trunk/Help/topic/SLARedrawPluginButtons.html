SLARedrawPluginButtons

プラグインボタンを再描画します。

構文

function SLARedrawPluginButtons(Name: PChar; No: Integer): BOOL; stdcall;

引数

Name プラグインの名前へのポインタ
No ボタン番号

戻り値

関数が成功したら TRUE 、失敗したら FALSE を返します。

解説

プラグインボタンを再描画します。

通常、関数 SLXButtonUpdate が呼ばれて TRUE を返さないと行われないプラグインボタンの再描画を、強制的に行わせることができます。

独自のタイミングでボタンの再描画を行いたいときに使用します。
