SLAGetPadTabWnd

パッドの ID から隠れている状態のパッドのウィンドウハンドルを取得します。

構文

function SLAGetPadTabWnd(ID: Integer): HWND; stdcall;

引数

ID 隠れている状態のウィンドウハンドルを取得するパッドの ID

戻り値

隠れている状態のパッドのウィンドウハンドルを返します。

解説

パッドの ID から隠れている状態のパッドのウィンドウハンドルを取得します。

パッドは隠れている状態と現れている状態では別のウィンドウになります。現れているパッドのウィンドウハンドルを取得するには SLAGetPadWnd を使う必要があります。
