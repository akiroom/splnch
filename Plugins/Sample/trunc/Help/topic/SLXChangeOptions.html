SLXChangeOptions

プラグインの設定ダイアログボックスを表示します。

構文

function SLXChangeOptions(hWnd: HWND): BOOL; stdcall;

引数

hWnd 設定ダイアログの親ウィンドウになるウィンドウのハンドル

戻り値

設定を変更したら TRUE 、設定を変更しなければ FALSE を返します。

TRUE を返した場合、SL4 は関数 SLXButtonUpdate を実行します。

解説

SL4 の［全体の設定］ダイアログボックスで、プラグインの設定を行う際に呼ばれます。

関数 SLXButtonOptions が実装されていない場合、SL4 の［ボタンの追加／変更］ダイアログボックスで、プラグインの設定を行う際にも呼ばれます。

プラグインの設定ダイアログボックスを表示して、プラグインの各種設定を行ってください。

TRUE を返すと各ボタンの再描画のために関数 SLXButtonUpdate が呼ばれます。

SL4 の対応
