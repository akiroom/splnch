SLXChangePadMouseEntered

パッドにマウスポインタが入ったときと出たときに呼ばれます。

構文

function SLXChangePadMouseEntered(hWnd: HWND, Entered: HWND): BOOL; stdcall;

引数

hWnd マウスが通過したパッドのウィンドウハンドル
Entered マウスが入ったときに TRUE、出たときに FALSE

戻り値

関数が成功したら TRUE 、失敗したら FALSE を返します。

解説

パッドにマウスポインタが入ったときと出たときに呼ばれます。すべてのパッドについて呼ばれます。

SL4 の対応

4.1.2 Build 264 2007/6/8 以降
