SLADeleteButton

ボタンを削除します。

構文

function SLADeleteButton(ID, GroupIndex, ButtonIndex: Integer): BOOL; stdcall;

引数

ID ボタンを削除するパッドの ID
GroupIndex 削除するボタンがあるボタングループのインデックス
ButtonIndex 削除するボタンの位置を示すインデックス

戻り値

関数が成功したら TRUE 、失敗したら FALSE を返します。

解説

既存のボタンを削除します。

4.0.0 Build 225 2000/12/22 以降
