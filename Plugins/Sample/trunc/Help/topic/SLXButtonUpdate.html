SLXButtonUpdate

ボタンのデータを更新する必要があるときに呼ばれます。

構文

function SLXButtonUpdate(No: Integer): BOOL; stdcall;

引数

No 更新するボタンの番号

戻り値

再描画の必要がある場合 TRUE 、ない場合は FALSE を返します。

解説

ボタンのデータを更新する必要があるときに呼ばれます。

関数 SLXGetButton で、定期的に呼ばれるように設定することが可能です。
