SLAGetPadID

パッドのウィンドウハンドルからパッドの ID を取得します。

構文

function SLAGetPadID(hWnd: HWND): Integer; stdcall;

引数

hWnd ID を取得するパッドのウィンドウハンドル

戻り値

パッドの ID を返します。

解説

パッドのウィンドウハンドルからパッドの ID を取得します。

パッドのウィンドウハンドルは起動ごとに毎回違いますが、ID は再起動後も有効です。

コールバック関数で取得できるパッドの情報はウィンドウハンドルのみですが、ID に変換することで、その他の Special Launch API を呼ぶのに役立ちます。
