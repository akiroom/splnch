SLXGetExplanation

バージョン情報に表示するプラグインの説明を返します。

構文

function SLXGetExplanation(Explanation: PChar; Size: Word): BOOL; stdcall;

引数

Explanation 説明を返すバッファを指すポインタ
Size Explanation が指すバッファのサイズ

戻り値

関数が成功したら TRUE 、失敗したら FALSE を返します。

解説

バージョン情報を表示する際に呼ばれます。

引数 Explanation が指すバッファに、バージョン情報に表示するプラグインの説明を返してください。バッファには、引数 Size が示すサイズ以上は入れないようにしてください。バッファーオーバーフローを起こします。
