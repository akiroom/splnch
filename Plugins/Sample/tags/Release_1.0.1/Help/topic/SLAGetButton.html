SLAGetButton

ボタンの情報を取得します。

構文

function SLAGetButton(ID, GroupIndex, ButtonIndex: Integer; Button: PSLAButton): BOOL; stdcall;

引数

ID ボタンの情報を取得するパッドの ID
GroupIndex ボタンの情報を取得するボタングループのインデックス
ButtonIndex ボタンの情報を取得するボタンのインデックス
Button ボタンの情報が入る構造体 TSLAButton へのポインタ

戻り値

関数が成功したら TRUE 、失敗したら FALSE を返します。

解説

ボタンの情報を引数 Button が指す構造体 Button に取得します。
