SLXSetInitFile

プラグインが使用できる設定ファイルを受け取ります。

構文

function SLXSetInitFile(InitFile: PChar): BOOL; stdcall;

引数

InitFile 設定ファイルの名前が入ったバッファを指すポインタ

戻り値

関数が成功したら TRUE 、失敗したら FALSE を返します。

解説

プラグインの起動時に呼ばれます。

引数 InitFile が指すバッファに、設定ファイルの名前が入っています。

通常は SL4 のデータフォルダに、プラグインの拡張子を .ini に変更したファイル名を受け取ります。

プラグインはこの設定ファイルの名前を記憶し、プラグインの起動時に呼ばれる関数 SLXBeginPlugin や、プラグインの終了時に呼ばれる関数 SLXEndPlugin などで適時設定を保存してください。

SL4 の対応

4.0.0 Build 191 2000/2/2 以降

SLXEndPlugin
