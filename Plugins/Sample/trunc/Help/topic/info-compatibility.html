旧バージョンとの互換性

SL4 本体のバージョンアップに伴い、新コールバック関数、新 Special Launch API が追加されます。

プラグインはこれらの新機能を使う場合に限り、SL4 本体のバージョンアップを必要とします。

4.0.0 Build 191 2000/2/2
プラグインのすべての関数、すべての構造体の名前を変更しました。この変更により、古いプラグインはすべて使えなくなります。
4.0.0 Build 204 2000/7/19
プラグインの仕様を変更しました。関数 SLXChangeOptions の引数に呼び出し元ウィンドウハンドルを追加しました。このため、古いプラグインは設定ダイアログを開く際にエラーが発生し止まることがあります。
4.0.0 Build 225 2000/12/22
関数 SLXButtonDrawEx を追加しました。
関数 SLXButtonCreate を追加しました。
関数 SLXButtonDestroy を追加しました。
Special Launch API を追加しました。
4.0.0 Build 226 2001/2/1
Special Launch API に SLAGetIcon を追加しました。
4.0.2 Build 240 2002/3/19
関数 SLXButtonOptions を追加しました。
