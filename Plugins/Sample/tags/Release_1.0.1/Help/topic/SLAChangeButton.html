SLAChangeButton

ボタンを変更します。

構文

function SLAChangeButton(ID, GroupIndex, ButtonIndex: Integer; Button: PSLAButton): BOOL; stdcall;

引数

ID ボタンを変更するパッドの ID
GroupIndex 変更するボタンがあるボタングループのインデックス
ButtonIndex 変更するボタンの位置を示すインデックス
Button 変更後のボタンの情報が入る構造体 TSLAButton へのポインタ

戻り値

関数が成功したら TRUE 、失敗したら FALSE を返します。

解説

既存のボタンを引数 Button が指す構造体 TSLAButton に置き換えます。
