SLAPasteButton

ボタンをクリップボードから貼り付けます。

構文

function SLAPasteButton(ID, GroupIndex, ButtonIndex: Integer): BOOL; stdcall;

引数

ID ボタンを貼り付けるパッドの ID
GroupIndex 貼り付けるボタングループのインデックス
ButtonIndex ボタンを貼り付ける位置を示すインデックス

戻り値

関数が成功したら TRUE 、失敗したら FALSE を返します。

解説

事前にクリップボードにコピーしておいたボタンを貼り付けます。
