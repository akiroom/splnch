SLAInsertButton

ボタンを挿入します。

構文

function SLAInsertButton(ID, GroupIndex, ButtonIndex: Integer; Button: PSLAButton): BOOL; stdcall;

引数

ID ボタンを挿入するパッドの ID
GroupIndex ボタンを挿入するボタングループのインデックス
ButtonIndex ボタンを挿入する位置のインデックス
Button ボタンの情報が入る構造体 TSLAButton へのポインタ

戻り値

関数が成功したら TRUE 、失敗したら FALSE を返します。

解説

新しいボタンを作成し、引数 ButtonIndex が示すインデックスの位置へ挿入します。
