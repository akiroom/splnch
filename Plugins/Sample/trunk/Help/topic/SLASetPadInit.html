SLASetPadInit

パッドのプロパティを 1 つセットします。

構文

function SLASetPadInit(ID: Integer; Key: PChar; Item: PChar): BOOL; stdcall;

引数

ID プロパティセットするパッドの ID
Key パッドのプロパティの名前
Buf プロパティの内容を入れるバッファ

戻り値

関数が成功したら TRUE 、失敗したら FALSE を返します。

解説

パッドのプロパティを 1 つセットします。

セットするプロパティは、引数 Key にパッドの設定ファイル PadX.ini と同じ名前を指定します。

セットするプロパティは引数 Buf に文字列として格納します。
