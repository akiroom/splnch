SLXButtonDestroy

ボタンの実体がパッド上から削除されると呼ばれます。

構文

function SLXButtonDestroy(No: Integer; hWnd: HWND; ButtonIndex: Integer): BOOL; stdcall;

引数

No ボタン番号
hWnd ボタンが作られるパッドのウィンドウハンドル
ButtonIndex 削除されるボタンの、ボタングループでのインデックス

戻り値

関数が成功したら TRUE 、失敗したら FALSE を返します。

解説

ボタンの実体がパッド上から削除されると呼ばれます。

SL4 はボタングループの切り替えやパッドのサイズ変更等でも、すべてのボタンは一旦削除され、再作成します。

SL4 の対応

4.0.0 Build 225 2000/12/22 以降
