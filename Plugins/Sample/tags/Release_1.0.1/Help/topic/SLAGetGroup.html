SLAGetGroup

ボタングループの情報を取得します。

構文

function SLAGetGroup(ID, GroupIndex: Integer; Group: PSLAGroup): BOOL; stdcall;

引数

ID ボタングループの情報を取得するパッドの ID
GroupIndex ボタングループのインデックス
Group ボタングループ情報が入る構造体 TSLAGroup へのポインタ

戻り値

関数が成功したら TRUE 、失敗したら FALSE を返します。

解説

ボタングループの情報を引数 Group が指す構造体 TSLAGroup に取得します。
