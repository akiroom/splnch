SLADeleteGroup

ボタングループを削除します。

構文

function SLADeleteGroup(ID, GroupIndex: Integer): BOOL; stdcall;

引数

ID 削除するボタングループがあるパッドの ID
GroupIndex 削除するボタングループのインデックス

戻り値

関数が成功したら TRUE 、失敗したら FALSE を返します。

解説

既存のボタングループを削除します。
