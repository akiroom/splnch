SLXButtonClick

ボタンがパッド上でクリックされると呼ばれます。

構文

function SLXButtonClick(No: Integer; hWnd: HWND): BOOL; stdcall;

引数

No 押されたボタンの番号
hWnd 押されたボタンがあるパッドのウィンドウハンドル

戻り値

関数が成功したら TRUE 、失敗したら FALSE を返します。

解説

ボタンがパッド上でクリックされると呼ばれます。

TRUE を返すと、ボタンが再描画されます。

SL4 の対応

4.0.0 Build 191 2000/2/2 以降
