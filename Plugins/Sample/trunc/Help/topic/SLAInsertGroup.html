SLAInsertGroup

新しいボタングループを挿入します。

構文

function SLAInsertGroup(ID, GroupIndex: Integer; Name: PChar): BOOL; stdcall;

引数

ID ボタングループを追加するパッドの ID
GroupIndex 新しいボタングループを挿入する位置のインデックス
Name 新しいボタングループの名前へのポインタ

戻り値

関数が成功したら TRUE 、失敗したら FALSE を返します。

解説

新しいボタングループを作成し、引数 GroupIndex が示すインデックスの位置へ挿入します。
