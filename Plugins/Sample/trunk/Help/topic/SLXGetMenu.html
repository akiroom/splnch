SLXGetMenu

プラグインに含まれるメニューを返します。

構文

function SLXGetMenu(No: Integer; MenuInfo: PSLXMenuInfo): BOOL; stdcall;

引数

No メニュー番号
MenuInfo メニュー情報を入れる構造体 TSLXMenuInfo へのポインタ

戻り値

引数 No の番号のメニューがある場合には TURE 、ない場合には FALSE を返します。

解説

プラグインの起動時に呼ばれます。

SL4 は引数 No を 0 から順に、この関数が FALSE を返すまで繰り返し実行します。

引数 MenuInfo が指す構造体 TSLXMenuInfo にメニューの情報を返してください。

SL4 の対応

4.0.0 Build 191 2000/2/2 以降
