SLXChangePadForeground

パッドのアクティブ状態が変更されたときに呼ばれます。

構文

function SLXChangePadForeground(hWnd: HWND, Foreground: HWND): BOOL; stdcall;

引数

hWnd アクティブ状態が変更されたパッドのウィンドウハンドル
Foreground アクティブ状態のとき TRUE、非アクティブ状態のとき FALSE

戻り値

関数が成功したら TRUE 、失敗したら FALSE を返します。

解説

パッドのアクティブ状態が変更されると呼ばれます。すべてのパッドについて呼ばれます。
