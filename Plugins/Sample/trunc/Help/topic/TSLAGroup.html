TSLAGroup

ボタングループの情報を定義する構造体です。

構文

type
  PSLAGroup = ^TSLAGroup;
  TSLAGroup = packed record
    PadID: Integer;
    GroupIndex: Integer;
    Name: array[0..1023] of Char;
    ButtonCount: Integer;
  end;

メンバ

PadID 所属するパッドの ID
GroupIndex 所属するパッドでのインデックス
Name ボタングループ名
ButtonCount ボタンの数
