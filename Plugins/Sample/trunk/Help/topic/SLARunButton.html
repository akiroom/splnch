SLARunButton

ボタンを実行します。

構文

function SLARunButton(ID: Integer; Button: PSLAButton): BOOL; stdcall;

引数

ID 実行したボタンの親ウィンドウになるパッドの ID
Button 実行するボタンの情報が入る構造体 TSLAButton へのポインタ

戻り値

関数が成功したら TRUE 、失敗したら FALSE を返します。

解説

引数 Button が指す構造体 TSLAButton のボタンを実行します。

4.0.0 Build 225 2000/12/22 以降
