SLAGetNextPadID

指定したパッドの次のパッドを取得します。

構文

function SLAGetNextPadID(ID: Integer): Integer; stdcall;

引数

ID 前のパッドの ID

戻り値

パッドの ID を返します。

解説

指定したパッドの次のパッドを取得します。

この順番は最近アクティブになったものから返ります。（Ctrl+Tab の順番と同じです。）

SLAGetPadCount で取得したパッドの回数分だけ繰り返せば、すべてのパッドを取得することができます。
