SLARenameGroup

ボタングループの名前を変更します。

構文

function SLARenameGroup(ID, GroupIndex: Integer; Name: PChar): BOOL; stdcall;

引数

ID 名前を変更するボタングループがあるパッドの ID
GroupIndex 名前を変更するボタングループのインデックス
Name 新しい名前へのポインタ

戻り値

関数が成功したら TRUE 、失敗したら FALSE を返します。

解説

既存のボタングループの名前を、引数 Name が指す名前に変更します。
