SLAChangePluginMenus

プラグインメニューを取得しなおします。

構文

function SLAChangePluginMenus(Name: PChar): BOOL; stdcall;

引数

Name プラグインの名前へのポインタ

戻り値

関数が成功したら TRUE 、失敗したら FALSE を返します。

解説

プラグインメニューを取得しなおします。

通常、プラグインの起動時にしか行われないプラグインのメニューの取得を、強制的に行わせることができます。

動的にプラグインのメニューを変化させたい場合に使用します。
